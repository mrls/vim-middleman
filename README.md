# vim-middleman

A middleman wrapper and some extra goodies.

_Note: Vim-Middleman is still under early development. This means you should
not rely on the commands and functions provided atm, but it also means you can
help drive the project ;)_

## Installation

Using [Plug](https://github.com/junegunn/vim-plug):

```
Plug 'mrls/vim-middleman'
```

## Documentation

See the plugin's documentation from Vim:

```
:h middleman
```
